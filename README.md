# GitPulse 🔍

A multi-page GitHub profile explorer built with HTML, Tailwind CSS, and JavaScript.

## Features
- Search any GitHub username
- View profile stats — followers, repos, gists
- Language breakdown chart (Chart.js)
- Browse and filter repositories by language, stars, and forks
- Recent searches saved locally

## Tech Used
HTML · Tailwind CSS · JavaScript · Chart.js · GitHub REST API

## Pages
- `index.html` — Search page
- `profile.html` — Profile overview + language chart
- `repos.html` — Repository list with filters

## Run
Just open `index.html` in your browser. No installation needed.

> Powered by GitHub REST API — no API key required
